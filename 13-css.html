CSS Externo (referencia, no aplicado aqui)

Para usar un archivo CSS externo se agrega en el head:

<link rel="stylesheet" href="styles.css">

Esto conecta la pagina con un archivo aparte (styles.css) que solo contiene reglas CSS, sin HTML. Es la forma mas usada en proyectos reales, porque permite cambiar el diseño de todo un sitio editando un solo archivo.
