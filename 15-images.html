Imagen de fondo en un parrafo

Este parrafo tiene una imagen de fondo aplicada con CSS (background-image), ademas de background-size:cover para que la imagen cubra todo el espacio sin deformarse.

Elemento picture (imagenes segun el dispositivo)

El navegador elige cual imagen mostrar segun el ancho de la pantalla. La etiqueta <img> al final es obligatoria, es la que se usa si el navegador no soporta <picture> o ninguna condicion coincide.
