Leccion 26 - El elemento Head HTML

Todo lo que aprendimos en esta leccion vive DENTRO de la etiqueta <head> de este mismo archivo (mira las lineas de arriba, entre <head> y </head>). Nada de eso se ve directamente en la pagina, pero cada elemento cumple una funcion importante.

title (obligatorio)

Mira la pestaña de tu navegador: dice "Leccion 26 - El elemento Head HTML", tomado directo del <title> de este archivo.

style

El bloque <style> en el head ya esta aplicando la fuente Arial a toda esta pagina, y el fondo gris de este texto:

Este parrafo tiene la clase "caja-codigo", definida en el <style> del head.

link (referencia externa, ejemplo)

Para conectar una hoja de estilos externa se usaria:

<link rel="stylesheet" href="mystyle.css">

meta (metadatos)

Este archivo ya incluye varios metadatos en el head, que no se ven en pantalla pero los usan navegadores y buscadores:

  • charset: define la codificacion de caracteres (UTF-8, para que tildes y ñ se vean bien)
  • description: descripcion de la pagina para buscadores
  • keywords: palabras clave relacionadas con el contenido
  • author: autor del documento
  • viewport: controla como se ve la pagina en dispositivos moviles

viewport (detalle importante)

La linea <meta name="viewport" content="width=device-width, initial-scale=1.0"> es esencial para que la pagina se vea bien en celulares y tablets, no solo en computadora. Se recomienda incluirla siempre.

base (ejemplo, no aplicado aqui)

Define una URL base por defecto para todos los enlaces relativos de la pagina. Ejemplo de como se veria en el head:

<base href="https://www.w3schools.com/" target="_blank">
