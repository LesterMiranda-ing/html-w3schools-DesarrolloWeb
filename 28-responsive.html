Leccion 28 - Responsive HTML

1. Meta viewport (obligatorio para que sea responsive)

Este archivo ya incluye en el head:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Sin esta linea, los celulares mostrarian la pagina como si fuera de escritorio (todo pequeño y con zoom).

7. Frameworks CSS (referencia, no aplicado aqui)

Frameworks como W3.CSS o Bootstrap ya traen sistemas de columnas responsive listos para usar, sin tener que escribir las media queries a mano. Se agregan enlazando su hoja de estilos:

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">

No los aplique en este archivo de practica para mantenerlo enfocado en HTML/CSS puro, pero es bueno saber que existen para proyectos mas grandes.
