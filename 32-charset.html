HTML Encoding (Character Sets)

Character Set Attribute

La especificación actual de HTML5 promueve el uso exclusivo de UTF-8 para dar soporte global a caracteres y símbolos.

Character Encoding Standards

ASCII (0-127): Soporta letras en inglés, números y caracteres básicos: a-z, A-Z, 0-9, !, $, +, -, (, ), @, <, >, ., #, ?

ANSI / ISO-8859-1 (160-255): Añade soporte para caracteres extendidos occidentales (coincidentes con UTF-8 en este rango).

UTF-8 (0-10000+): Cubre prácticamente todos los alfabetos y símbolos existentes.
